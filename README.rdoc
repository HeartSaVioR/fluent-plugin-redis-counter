= Redis counter plugin for fluent

fluent-plugin-redis-counter is a fluent plugin to count-up/down redis keys.

== Configuration

    <match redis_counter.**>
      type redis_counter

      host localhost
      port 6379

      # database number is optional.
      db_number 0        # 0 is default
    </match>
